# Analyse des campagnes Quickstarter (2022-2024)

Ce projet contient l'analyse des campagnes de financement participatif sur Kiickstarter pour les années 2022 à 2024.

## Structure du dépôt

- `notebooks/` : contient les notebooks
  - `01_quickstarter_data_preprocessing.ipynb` : nettoyage et préparation des données
  - `02_quickstarter_analysis_2022_2024.ipynb` : exploration et visualisation des données
